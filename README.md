# youzan-rider

## 简介
本项目为VueCLI项目，需要在本机安装node.js，然后按照下面的步骤运行

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
